# 配置相关

## IOS 上下拉白屏

```json
"globalStyle": {
	 // ...
	"app-plus": {
		"bounce": "none"
	}
},
```

## 遮罩层滚动穿透

`@touchmove.native.prevent` 设置在 遮罩层上

## sdk 警告

```json
"app-plus" : {
    "compatible" : {
        ignoreVersion" : true //忽略版本检查提示框，HBuilderX1.9.0及以上版本支持
    }
}
```
